# webrtc_sample
WebRTC sample application.

# Rrequire
- node.js
- npm

# Set Up
```sh
$ sudo apt-get update
$ sudo apt-get install nodejs npm
$ sudo update-alternatives --install /usr/bin/node node /usr/bin/nodejs
$ sudo npm install -g bower

$ git clone git@github.com:umatoma/webrtc_sample.git
$ cd webrtc_sample
$ npm install
$ bower install
```

# Start App
```sh
$ npm start
```
